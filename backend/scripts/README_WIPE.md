# Как удалить все данные из базы

## Вариант 1: Очистить данные (оставить схему и пользователей)

Остановите бэкенд, затем выполните SQL-скрипт.

**Путь к базе по умолчанию:** `backend/dayzsmartcf.db` (или из переменной `DATABASE_URL` в `.env`).

### Windows (PowerShell)

```powershell
cd backend
sqlite3 dayzsmartcf.db ".read scripts/wipe_data.sql"
```

Если `sqlite3` не установлен, скачайте [sqlite-tools](https://www.sqlite.org/download.html) и добавьте в PATH, либо используйте Вариант 2.

### Linux / macOS

```bash
cd backend
sqlite3 dayzsmartcf.db < scripts/wipe_data.sql
```

После этого в базе не останется игроков, групп, истории и отслеживания. **Логин (users) сохранится.**

---

## Вариант 2: Удалить файл базы полностью

1. Остановите бэкенд.
2. Удалите файл базы:
   - по умолчанию: `backend/dayzsmartcf.db`;
   - если в `.env` указан другой путь — удалите его.
3. При необходимости удалите журнал: `dayzsmartcf.db-journal`.
4. Запустите бэкенд снова — миграции создадут пустые таблицы.

**Важно:** при этом таблица пользователей (users) будет пустой. Чтобы снова войти, задайте в `.env` `ADMIN_USER` и `ADMIN_PASS` — при первом запуске создастся учётка админа.
